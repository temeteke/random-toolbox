# 🎲 Random Toolbox

シンプルで使いやすいランダム生成ツールボックスです。PWA対応で、オフラインでも快適に動作します。

## 機能

### 🔢 乱数生成
- 指定範囲内のランダムな数字を生成
- 生成履歴の保存と表示

### 📋 リスト選択
- リストから項目をランダムに選択
- 入力した項目リストは自動保存
- 選択履歴の保存と表示

### 🎰 ルーレット
- ビジュアルな回転ルーレット
- 複数項目から抽選
- SVGアニメーションによる滑らかな演出

### 🎲 サイコロ
- 複数個のサイコロを同時に振れる（1〜10個）
- 様々な面数に対応（D6、D8、D10、D12、D20、D100）
- 合計値と個別の出目を表示
- TRPG・ボードゲームに最適

### 🃏 カードドロー
- トランプカードのランダム抽出
- 1〜52枚から選択可能
- ジョーカー含む/含まない設定
- 引いたカードの履歴表示

### 🎨 カラーピッカー
- ランダムな色を生成（HEX、RGB表示）
- カラーパレット生成（調和のとれた5色セット）
- ワンクリックでコピー機能
- お気に入り色の保存機能

### 📅 日付ランダマイザー
- 指定期間内のランダムな日付生成
- 過去/未来の範囲指定可能
- 曜日フィルター（特定の曜日のみ選択）
- イベントプランニングや抽選に便利

### ✨ パスワード生成器
- 文字数指定可能（8〜128文字）
- 文字種別選択（大文字、小文字、数字、記号）
- 強度インジケーター表示
- ワンクリックコピー機能
- セキュアなパスワード生成

### 🎯 決定ツール
- シンプルなYes/No決定
- カスタム選択肢（2〜5個）設定可能
- 迷った時の意思決定支援

### 🔀 シャッフラー
- リスト全体をランダムに並び替え
- グループ数指定で自動分割機能
- チーム分けやプレゼン順決めに便利
- シャッフル結果の履歴保存

### 🎊 抽選ボックス
- ビンゴ風の抽選システム
- 一度選ばれた項目は除外
- 残り項目数の表示
- リセット機能
- イベントやプレゼント抽選に最適

### その他の機能
- 📱 PWA対応（オフラインで動作可能）
- 🎨 レスポンシブデザイン
- 💾 ローカルストレージによるデータ永続化
- 🔄 タブで機能を切り替え
- 📜 全機能で履歴保存対応（最大50件）

## GitHub Pagesへのデプロイ方法

1. GitHubリポジトリの「Settings」タブを開く
2. 左サイドバーの「Pages」をクリック
3. 「Source」セクションで「Deploy from a branch」を選択
4. ブランチを選択（mainまたはmaster）してルートディレクトリ「/(root)」を選択
5. 「Save」をクリック

数分後、`https://<username>.github.io/<repository-name>/` でアプリにアクセスできます。

## ローカルでの実行

```bash
# シンプルなHTTPサーバーを起動
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。
