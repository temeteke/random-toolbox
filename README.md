# 🎲 Random Toolbox

シンプルで使いやすいランダム生成ツールボックスです。PWA対応で、オフラインでも快適に動作します。

## 機能

### 基本ツール

#### 🔢 乱数生成
- 指定範囲内のランダムな数字を生成
- 生成履歴の保存と表示

#### 📋 リスト選択
- リストから項目をランダムに選択
- 入力した項目リストは自動保存
- 選択履歴の保存と表示

#### 🎰 ルーレット
- ビジュアルな回転ルーレット
- 複数項目から抽選
- SVGアニメーションによる滑らかな演出

#### 🎲 サイコロ
- 複数個のサイコロを同時に振れる（1〜10個）
- 様々な面数に対応（D6、D8、D10、D12、D20、D100）
- 合計値と個別の出目を表示
- TRPG・ボードゲームに最適

#### 🃏 カードドロー
- トランプカードのランダム抽出
- 1〜52枚から選択可能
- ジョーカー含む/含まない設定
- 引いたカードの履歴表示

#### 🎨 カラーピッカー
- ランダムな色を生成（HEX、RGB表示）
- カラーパレット生成（調和のとれた5色セット）
- ワンクリックでコピー機能
- お気に入り色の保存機能

#### 📅 日付ランダマイザー
- 指定期間内のランダムな日付生成
- 過去/未来の範囲指定可能
- 曜日フィルター（特定の曜日のみ選択）
- イベントプランニングや抽選に便利

#### ✨ パスワード生成器
- 文字数指定可能（8〜128文字）
- 文字種別選択（大文字、小文字、数字、記号）
- 強度インジケーター表示
- ワンクリックコピー機能
- セキュアなパスワード生成

#### 🎯 決定ツール
- シンプルなYes/No決定
- カスタム選択肢（2〜5個）設定可能
- 迷った時の意思決定支援

#### 🔀 シャッフラー
- リスト全体をランダムに並び替え
- グループ数指定で自動分割機能
- チーム分けやプレゼン順決めに便利
- シャッフル結果の履歴保存

#### 🎊 抽選ボックス
- ビンゴ風の抽選システム
- 一度選ばれた項目は除外
- 残り項目数の表示
- リセット機能
- イベントやプレゼント抽選に最適

### 新機能（拡張ツール）

#### 🪙 コイントス
- シンプルな表/裏の2択
- アニメーション付き演出
- 統計表示（表/裏の出現率）

#### 🍔 ランダムメニュー提案

- カテゴリー別（和食、洋食、中華、ファストフード、ヘルシー）
- カスタムメニュー登録可能

#### 📛 名前ジェネレーター

- 日本語名（姓名）
- 英語名（フルネーム）
- ユーザー名生成
- プロジェクト名生成
- 性別・長さの設定可能

#### 🎬 ビンゴカード生成
- カスタマイズ可能なビンゴカード
- 3×3、4×4、5×5サイズ対応
- 数字またはカスタム項目
- 5×5カードはFREEマス自動配置
- イベントやパーティーに最適

#### 📝 Lorem Ipsum生成器

- 日本語・英語対応

#### ⚖️ 重み付き抽選
- 各項目に確率（重み）を設定できる抽選
- 確率一覧表示
- パーセンテージ自動計算

#### ⏱️ ランダムタイマー
- ランダムな時間でタイマー設定
- 時間範囲指定（分単位）
- 一時停止・再開機能
- プログレスバー表示

#### 🌈 グラデーション生成器
- ランダムなグラデーション生成
- 線形・放射状・円錐形対応
- 2〜5色のグラデーション
- 角度調整可能
- CSS/SVGコードの出力
- ワンクリックコピー
- デザイン作業に便利

#### 🗺️ 座標ランダマイザー
- ランダムな緯度経度生成
- プリセット範囲（世界、日本、東京、アメリカ、ヨーロッパ、アジア）
- カスタム範囲指定可能
- Google Mapsで直接開く機能
- 座標コピー機能

#### ✏️ タスクピッカー
- TODOリストからランダムに選択
- 優先度設定（高・中・低）

- 完了タスク管理
- タスク管理の新しいアプローチ

## 技術スタック

### フロントエンド
- **HTML5** - セマンティックなマークアップ
- **CSS3** - モダンなスタイリングとアニメーション
- **JavaScript (ES6+)** - バニラJavaScriptによる実装

### フレームワーク・ライブラリ
- **Alpine.js 3.x** - 軽量なリアクティブフレームワーク
  - 宣言的なデータバインディング
  - コンポーネントベースのアーキテクチャ
  - 軽量（〜15KB gzip）でパフォーマンスに優れる

### PWA技術
- **Service Worker** - オフライン対応とキャッシュ戦略
- **Web App Manifest** - ホーム画面追加とアプリ風UI
- **Cache API** - 静的アセットのキャッシング

### ブラウザAPI
- **Local Storage API** - データの永続化（履歴、設定など）
- **Clipboard API** - パスワードや色コードのコピー機能
- **Canvas/SVG** - ルーレットなどの視覚的要素

### 特徴
- ✅ ビルドツール不要（Zero Build）
- ✅ フレームワークの依存性が最小限
- ✅ 高速な読み込みとレスポンス
- ✅ モダンブラウザ対応
- ✅ モバイルフレンドリー

## GitHub Pagesへのデプロイ方法

1. GitHubリポジトリの「Settings」タブを開く
2. 左サイドバーの「Pages」をクリック
3. 「Source」セクションで「Deploy from a branch」を選択
4. ブランチを選択（mainまたはmaster）してルートディレクトリ「/(root)」を選択
5. 「Save」をクリック

数分後、`https://<username>.github.io/<repository-name>/` でアプリにアクセスできます。

## ローカルでの実行

```bash
# シンプルなHTTPサーバーを起動
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。
